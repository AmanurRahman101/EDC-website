# EDC Website

Welcome to the EDC Website project! This repository contains the source code for the EDC website.

## Getting Started

To get started, make sure you open this folder as your active workspace in your IDE.

## Directory Structure

- `index.html` - The main HTML entry point.
- `css/` - Custom CSS styling files.
- `js/` - Frontend JavaScript logic files.
- `assets/` - Image, font, and other media assets.
